# 仁家坪新村网站

水墨田园风格的村级门户网站

## 部署方式

### Vercel 一键部署

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/village-website)

## 本地预览

```bash
cd village-website
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 项目结构

```
village-website/
├── index.html          # 首页
├── news/               # 新闻目录
├── images/             # 图片资源
│   ├── carousel-*.jpg  # 轮播图
│   ├── members/        # 村支两委照片
│   └── ...
├── vercel.json         # Vercel配置
└── README.md
```

## 技术栈

- 纯HTML/CSS/JavaScript
- 水墨田园风格设计
- 响应式布局
