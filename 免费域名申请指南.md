# 仁家坪新村免费域名申请指南

## 推荐方案（按推荐度排序）

### 方案一：GitHub Pages（最推荐 ✅）
- **免费域名**：`你的用户名.github.io` 或 `你的用户名.github.io/renjiaping`
- **免费托管**：GitHub Pages 提供 100% 免费静态网站托管
- **步骤**：
  1. 注册 GitHub 账号 → https://github.com/signup
  2. 创建仓库，命名为 `renjiaping` 或 `你的用户名.github.io`
  3. 将 `index.html`、`style.css`、`script.js` 上传到仓库
  4. 进入仓库 Settings → Pages → 选择分支 main → Save
  5. 几分钟后即可访问 `https://你的用户名.github.io/renjiaping`

### 方案二：Vercel（推荐 ✅）
- **免费域名**：`renjiaping.vercel.app`（可自定义前缀）
- **免费托管**：Vercel 免费套餐足够使用
- **步骤**：
  1. 注册 Vercel → https://vercel.com/signup
  2. 从 GitHub 导入仓库，或直接拖拽文件部署
  3. 自动分配 `xxx.vercel.app` 域名

### 方案三：Netlify
- **免费域名**：`renjiaping.netlify.app`
- **步骤类似 Vercel**，注册 → https://netlify.com → 拖拽部署

### 方案四：.eu.org 免费域名
- **免费域名**：如 `renjiaping.eu.org`
- 需要自有服务器或配合 GitHub Pages/CNAME 使用
- 申请地址 → https://nic.eu.org
- 审核周期约 1-2 周

### 方案五：Cloudflare Pages
- **免费域名**：`renjiaping.pages.dev`
- 注册 → https://pages.cloudflare.com → 连接 GitHub 仓库自动部署
